Text Formatting Elements

These are used to display special types of text for better look and feel without any additional CSS.

This contains bold text.

This contains strong text, which tells the browser that the text is important, which is typically displayed in bold.

This contains italic text.

This contains emphasized text, which tells that browser that the text need special attention, which is displayed in italic.

This contains underline text.

This contains strike through text, and it is not supported in HTML5.

This contains small text, which is usually one unit smaller than the base font size.

This contains big text, which is usually one unit bigger than the base font size.

This contains marked text, which is usually highlighted with yellow background.

This contains deleted text.

This contains inserted text.

This contains subscript text.

This contains superscript text.

This contains monospaced (example im19) text, where all letters take the same width as seen in early computers and typewriters. It is not supported in HTML5.
